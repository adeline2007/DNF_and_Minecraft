# DNF_and_Minecraft
In the program, the user fills in the truth table. After that, the program simplifies it and presents it as a DNF. After that you can convert the schematic to WorldEdit Minecraft schematic format. The program is written in Python, as well as in Russian.
Details:
At the very beginning, we are asked to enter the number of inputs per circuit (Maximum 8)
Then we fill in the truth table, there are two buttons to help: Fill in all the blank 0, or 1
After filling, click the "simplify" button. We are starting to see schema and DNF
In the upper right corner we can enter the file name and convert it to schematic format.
By default it is done for Tlaunther on drive C. The path to the folder with schematics can be changed in line 184 of the file "main_mine_schem"

В программе пользователь заполняет таблицу истинности. После этого программа упрощает её и представляет в виде ДНФ. После этого вы можете преобразовать схему в формат схематика для WorldEdit Minecraft. Программа написана на Python, а также на русском языке. 
Подробности:
В самом начале нас просят ввести колличество воходов на схему (Максимум 8)
Затем мы заполняем таблицу истинности, в помощь есть две кнопки: Заполнить всё незаполненное 0, либо же 1
После заполнения нажимаем кнопку "упростить". Мы начинаем видеть схему и ДНФ
В правом верхнем углу мы можем ввести наззвание файла и конвертировать его в формат схематика.
По умолчаниию сделано для Tlaunther на диске C. Путь к папке со схематиками можно изменить в 184 строке файла "main_mine_schem"
